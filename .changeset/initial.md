First public release of `@fluent-kit/ui`.

Includes:
- Config-driven components: `DataTable`, `Modal`, `Input`, `Select`, `FormField`
- Primitives: `Button`, `Badge`, `Tag`, `Card`, `Checkbox`, `Switch`, `Tooltip`, `Popover`, `Tabs`, `DropdownMenu`, `Sheet`, `Separator`, `Skeleton`, `Textarea`
- Composite composites: `SidebarNav`, `SegmentedControl`, `DetailDrawer`, `DateRangePopover`
- RHF adapter subpath `@fluent-kit/ui/rhf` (`FormFieldController`)
- Compiled CSS with bundled Geist font — zero Tailwind configuration required
- Full TypeScript types with antd-shaped API (TanStack Table internals hidden)
- Light/dark theming via CSS variables and `dark` class on `<html>`
